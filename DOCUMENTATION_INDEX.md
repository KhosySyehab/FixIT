## üéØ Key Takeaways

### What FixIT Does
- Allows users to report public facility issues
- Tracks reports with location & severity
- Auto-calculates priority based on votes & severity
- Admin dashboard for management
- Gamification to encourage participation

### What FixIT Uses
- Node.js + Express backend
- React + Vite frontend
- MongoDB database
- Leaflet for maps
- Tailwind CSS for styling

### What FixIT Provides
- ‚úÖ User authentication (JWT)
- ‚úÖ Report creation & management
- ‚úÖ Voting & prioritization
- ‚úÖ Gamification (points, levels, badges)
- ‚úÖ Admin dashboard & analytics
- ‚úÖ Email notifications
- ‚úÖ Interactive maps
- ‚úÖ Complete audit trail

### How to Get Started
1. Clone/download project
2. Install dependencies (backend & frontend)
3. Configure .env files
4. Run seed script
5. Start servers
6. Login & explore
